⚙️ Configuration Firebase
Pour mettre le logiciel en ligne avec des données partagées, vous avez besoin d'un compte Firebase gratuit (Google). Suivez ces étapes :
1
Allez sur console.firebase.google.com et connectez-vous avec votre compte Google.
2
Cliquez "Créer un projet" → nommez-le triomphant-mmb → Créer.
3
Dans le projet, cliquez "Ajouter une application Web" (icône </>) → nommez-la → Enregistrer → copiez la configuration.
4
Dans le menu gauche → Firestore Database → "Créer une base de données" → Mode test → Suivant → Activer.
5
Menu gauche → Authentication → "Commencer" → Fournisseurs de connexion → Anonyme → Activer.
6
Collez vos informations Firebase ci-dessous et cliquez "Connecter".

📋 Comment importer depuis Access ou Excel
Le logiciel accepte les fichiers .xlsx (Excel) et .csv (Access, LibreOffice, etc.)
📊 Depuis Microsoft Excel
1. Ouvrez votre fichier Excel
2. Assurez-vous que la 1ère ligne contient les en-têtes de colonnes
3. Enregistrez directement le fichier .xlsx
4. Cliquez "Importer Excel/CSV" et sélectionnez ce fichier
🗄️ Depuis Microsoft Access
1. Ouvrez votre base Access
2. Clic droit sur votre table clients → Exporter → Fichier texte (.csv)
3. Cochez "Première ligne contient les noms de champs"
4. Enregistrez → cliquez "Importer Excel/CSV"
